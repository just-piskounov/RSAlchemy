# RSAlchemy

RSAlchemy is a cryptographic project focused on implementing and exploring RSA encryption and decryption. The project includes a GUI for ease of use and supports various functionalities, including encryption, decryption, and cryptographic attack demonstrations.

## Features

- **RSA Implementation**:
  - Full RSA encryption and decryption support.
  - Multiple padding schemes (e.g., PKCS#1, OAEP).

- **Cryptographic Attacks**:
  - Educational demonstrations of attacks on RSA, such as:
    - Common modulus attack.
    - Low exponent attack.
    - Factorization-based attacks.

- **File Encryption**:
  - Encrypt and decrypt files, including text and images.

- **Graphical User Interface (GUI)**:
  - Intuitive GUI for easy interaction and demonstration of RSA functionalities.

- **Examples and Tests**:
  - Pre-built examples and tests to understand RSA concepts in action.
